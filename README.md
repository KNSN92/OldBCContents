# OldBCContents

このmodは、1.7.10のBuildCraftに昔のバージョンのBuildCraftの要素の一部を復活させるmodです。具体的には以下の要素を復活させます。

- 隣のチェストのアイテムを使用してクラフトができる全自動作業台
- エネルギーパイプに接続出来るレッドストーンエンジン(木エンジン)

現状はこの2つのみを追加しますが、後々には他にもいくつかの要素を追加することを考えています。気が向けばですが。
